# go-unit-of-work

Demonstrates the unit of work pattern using the library [github.com/alextanhongpin/uow](https://github.com/alextanhongpin/uow).

## How to Start

```bash
# Run postgres in docker container.
$ make up

# Run the main program.
$ make start

# Stop the docker container.
$ make down

# Run the test.
$ make test
```
